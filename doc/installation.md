# Installation

## Requirements

- Python 3.10 or higher
- pytest 9.0 or higher

## Installing from PyPI

The easiest way to install **pytest-pyspec** is from PyPI:

```bash
pip install pytest pytest-pyspec
```

This will automatically install pytest if it's not already installed.

## Installing from Source

If you want to install the latest development version:

```bash
git clone https://github.com/felipecrp/pytest-pyspec.git
cd pytest-pyspec
pip install -e .
```

## Verifying Installation

To verify that pytest-pyspec is installed correctly:

```bash
pytest --version
```

You should see pytest-pyspec listed in the plugins section.

## Upgrading

To upgrade to the latest version:

```bash
pip install --upgrade pytest-pyspec
```

## Uninstalling

To remove pytest-pyspec:

```bash
pip uninstall pytest-pyspec
```
